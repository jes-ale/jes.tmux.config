# jes.tmux.config
### personal tmux config
```bash
sudo apt install tmux
git clone https://github.com/jes-ale/jes.tmux.config.git ~/.config/tmux
```
setup tpm
```bash
git clone https://github.com/tmux-plugins/tpm ~/.config/tmux/plugins/tpm
```
fetch plugins
```bash
prefix + I
```
Source tmux conf
```bash
prefix + r
```
